# KagOJ



## How to start

#### Installation

At first clone this repository by

`git clone https://github.com/TamimEhsan/KagOJ.git`

#### Start Backend

Then go into `Kagoj/Kagoj Backend` 

and install dependencies and then start node server

```
npm i
npm run dev
```



#### Start Frontend

Then go into `Kagoj/Kagoj Frontend` 

and install dependencies and then start node server

```
npm i
npm start
```
